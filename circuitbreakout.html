Circuit Breakout: Dual Polarity

A commercial Steam action title built around a polarity-based combat system emphasizing systemic gameplay and scalable architecture.

Project Overview

Circuit Breakout is a fast-paced action game centered around a polarity-switching combat mechanic. Players strategically shift between two energy states to counter enemies and optimize combat flow.

My Responsibilities

  • Designed and implemented dual polarity combat system
  • Built modular ability framework
  • Developed AI state-based behavior architecture
  • Implemented combat feedback systems
  • Optimized gameplay systems

Core Systems

Dual Polarity System

The core mechanic allows switching between two energy states to counter specific enemy types. Designed using modular state architecture to ensure scalability.

Ability Framework

Implemented reusable ability architecture supporting cooldowns, conditional triggers, and layered effects.

Enemy AI

Built scalable state-machine AI allowing extension without rewriting base behavior logic.
