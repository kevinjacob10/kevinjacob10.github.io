Project Overview

OpenGenLab is an immersive virtual reality learning environment designed for first-year biology students. Funded by the Government of Ontario and supported by eCampusOntarioâ€™s Virtual Learning Strategy, the project simulates real molecular biology laboratory workflows through interactive VR tools.

Students explore core genetics concepts using three primary laboratory tools: a Virtual Microscope, a PCR Thermal Cycler and a DNA Sequencer. The experience blends guided instruction with puzzle-based gameplay - requiring players to correctly execute real experimental procedures such as DNA isolation, amplification and sequencing.

Core Systems

VR Gaze Interaction System

Head-tracked gaze interaction framework enabling tool selection, menu activation, object manipulation and procedural validation without traditional controllers.

PCR Thermal Cycler Simulation

Puzzle-driven sequencing system requiring players to correctly arrange buffers, reagents and execution order before initiating the amplification process. Includes state validation and error feedback logic.

DNA Sequencer Logic System

Implemented primer matching and DNA sequence validation system where players analyze plant samples, select correct primers and interpret sequencing output. Designed to reflect real-world experimental logic.
